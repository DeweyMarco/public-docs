# Welcome to Example API

Example API gives developers access to state-of-the-art language models through a simple,
familiar REST interface.

## What you can build

- **Chatbots and assistants** — conversational interfaces for your app
- **Content generation** — drafts, summaries, translations, and more
- **Code tools** — autocompletion, explanation, review
- **Data extraction** — structured output from unstructured text

## How it works

You send a prompt (and optional configuration) to the API. We run it through the model
and return the generated text. That's it.

```json
POST /v1/completions

{
  "model": "example-v1",
  "prompt": "Summarize this in one sentence: ...",
  "max_tokens": 100
}
```
